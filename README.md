# Start Project

### Dev

```bash
npm run dev
```

### Production

```bash
npm run install
npm run build
npm run start:prod
```

# ENV

```
PORT=           # Server Port
DB_USER=        # Database User
DB_PASSWORD=    # Database Password
DB_NAME=        # Database Name
DB_HOST=        # Database Host
DB_PORT=        # Database Port
access_token=   # Secret key
```
